Introducción

El estudio de funciones polinómicas mediante el cálculo diferencial permite comprender su comportamiento gráfico y algebraico. Este proyecto tiene como propósito analizar tres funciones aplicando herramientas del cálculo diferencial, como derivadas primera y segunda, para identificar puntos críticos, intervalos de crecimiento y decrecimiento, concavidad y puntos de inflexión. Se simula el lanzamiento de un cohete para interpretar los resultados matemáticos en un contexto físico real.

Objetivos

1. Aplicar las reglas de derivación para analizar el comportamiento de funciones polinómicas.

2. Determinar puntos críticos, intervalos de crecimiento y decrecimiento, concavidad y puntos de inflexión.

3. Interpretar los resultados en el contexto físico de la trayectoria de un cohete.

Problema Inicial

Se desea analizar el comportamiento de funciones que modelan el movimiento de un cohete en el aire. A partir de estas funciones, se busca determinar puntos críticos, intervalos de crecimiento y decrecimiento, puntos de inflexión y concavidad. Además, se pretende encontrar el ángulo de lanzamiento óptimo, la altura máxima alcanzada, la distancia horizontal máxima y el tiempo de vuelo.

Metodología

Se seleccionaron funciones polinómicas representativas. A cada función se le aplicó la primera derivada para hallar puntos críticos y determinar crecimiento/decrecimiento. Posteriormente, se usó la segunda derivada para encontrar puntos de inflexión y concavidad. Se interpretaron los resultados matemáticos en un contexto físico real. Finalmente, se compararon los resultados con entregas anteriores para obtener conclusiones más completas.

Análisis de g(x) = -2x³ + 12x² - 18x + 5

Análisis de h(d) = -0.5d² + 6d

Interpretación Física

Conclusiones

El cálculo diferencial permite analizar funciones que modelan fenómenos reales como la trayectoria de un cohete. A través de las derivadas, se identificaron comportamientos clave como puntos críticos, máximos, mínimos, concavidad y tiempo de vuelo. La comparación con entregas anteriores muestra cómo se puede refinar y ampliar un modelo matemático aplicado.
